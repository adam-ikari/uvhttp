\doxysubsection{Detailed Description}
UVHTTP 可配置常量定义 

本文件集中了所有可配置的常量，按功能模块分类。

==================== 配置指南 ====================


\begin{DoxyEnumerate}
\item 编译时配置 通过 CMake 编译选项或直接修改本文件中的宏定义来配置 推荐使用 CMake 选项，便于维护和版本控制
\item 性能调优建议
\begin{DoxyItemize}
\item 小型应用（\texorpdfstring{$<$}{<} 1000 RPS）：使用默认配置
\item 中型应用（1000-\/10000 RPS）：增加连接数和缓冲区大小
\item 大型应用（\texorpdfstring{$>$}{>} 10000 RPS）：需要系统调优，参考下面的大型应用配置
\end{DoxyItemize}
\item 内存优化
\begin{DoxyItemize}
\item 内存受限环境：减小缓冲区大小，禁用缓存
\item 高性能环境：增加缓冲区大小，启用所有缓存
\end{DoxyItemize}
\end{DoxyEnumerate}
