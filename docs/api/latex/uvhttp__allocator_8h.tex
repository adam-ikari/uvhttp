\doxysubsection{Detailed Description}
Unified memory allocator with compile-\/time optimization. 

This module provides a unified memory allocation interface that can be configured at compile time to use either system allocator or mimalloc. All functions are inline-\/optimized for zero runtime overhead.

\begin{DoxyNote}{Note}
Allocator type is selected at compile time via UVHTTP\+\_\+\+ALLOCATOR\+\_\+\+TYPE 

All functions are inline for zero runtime overhead 
\end{DoxyNote}
