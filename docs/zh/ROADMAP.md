# UVHTTP 路线图

## 愿景

UVHTTP 旨在成为最受信任、性能最高、最友好的 C 语言 HTTP 服务器库，为生产级、零开销的网络解决方案树立标准。

## 当前状态 (v2.5.0)

### ✅ 已完成
- 高性能 HTTP/1.1 服务器 (23,226 RPS)
- 支持全双工通信的 WebSocket
- 零拷贝文件传输
- 带预热的 LRU 缓存
- 通过 mbedtls 支持 TLS 1.3
- 32 位嵌入式系统支持
- 完整的文档
- 基于 Python 的配置工具
- 输入验证和安全加固
- 企业级错误处理

### 🔄 进行中
- 增强测试覆盖率（目标：80%）
- 平台扩展（macOS、Windows）
- 性能优化
- 社区参与

## 技术栈演进

### 当前技术栈
- **核心**: C11、libuv、llhttp
- **TLS**: mbedtls
- **内存**: mimalloc（可选）
- **哈希**: xxHash
- **JSON**: cJSON（可选）
- **测试**: Google Test

### 未来考虑
- **TLS**: BoringSSL、LibreSSL
- **哈希**: FarmHash、MetroHash
- **内存**: jemalloc、tcmalloc
- **异步**: io_uring（Linux）、kqueue（BSD）
- **压缩**: zstd、brotli
