)

Nginx 优化：使用 sendfile 零拷贝发送静态文件（混合策略）

根据文件大小自动选择最优策略：

  • 小文件 (< 4KB): 使用传统方式（避免 sendfile 开销）
  • 中等文件 (4KB - 10MB): 使用异步 sendfile
  • 大文件 (> 10MB): 使用分块 sendfile
Parameters
file_path文件路径
responseHTTP响应
Returns
UVHTTP_OK成功，其他值表示失败
