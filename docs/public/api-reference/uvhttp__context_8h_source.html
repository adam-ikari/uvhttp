40 /* ============ 内存分配器说明 ============ */
41 /*
42  * UVHTTP 内存分配器采用编译时宏设计，零开销抽象
43  *
44  * 不使用运行时分配器提供者接口，原因：
45  * 1. 性能优先：避免函数指针调用开销
46  * 2. 编译时优化：编译器可以内联和优化分配调用
47  * 3. 简单直接：减少复杂性，提高可维护性
48  *
49  * 内存分配器类型通过 UVHTTP_ALLOCATOR_TYPE 编译宏选择：
50  * - 0: 系统默认分配器 (malloc/free)
51  * - 1: mimalloc 高性能分配器
52  * - 2: 自定义分配器 (外部链接)
53  *
54  * 使用方式：
55  * #include "uvhttp_allocator.h"
56  * void* ptr = UVHTTP_MALLOC(size);
57  * UVHTTP_FREE(ptr);
58  *
59  * 详见 uvhttp_allocator.h 中的详细说明
60  */
