# UVHTTP Brand Guidelines

## Brand Identity

### Vision
To be the most trusted, performant, and developer-friendly HTTP server library for C applications.

### Mission
Providing a production-grade, zero-overhead HTTP/1.1 and WebSocket server library that empowers developers to build high-performance applications with confidence.

### Core Values
- **Performance First**: Zero-cost abstractions, maximum throughput
- **Security Built-in**: Input validation, buffer overflow protection, TLS support
- **Developer Experience**: Clean APIs, comprehensive documentation, helpful tools
- **Production Ready**: Robust error handling, complete testing, enterprise-grade quality
- **Minimalist Design**: Focus on core functionality, avoid unnecessary complexity

## Visual Identity

### Primary Colors
- **Primary Blue**: `#2563eb` - Trust, stability, technology
- **Secondary Purple**: `#7c3aed` - Innovation, creativity
- **Accent Cyan**: `#06b6d4` - Modern, forward-looking

### Color Usage
- Primary: Headers, navigation, primary actions
- Secondary: Gradients, highlights, emphasis
- Accent: Links, icons, decorative elements

### Typography
- **Headings**: Inter, bold, 600 weight
- **Body**: Inter, regular, 400 weight
- **Code**: JetBrains Mono, monospace
- **Numbers**: Tabular nums for data

### Logo Guidelines
- Use SVG format for scalability
- Maintain minimum padding of 2x logo height
- Don't stretch or distort
- Use on light or dark backgrounds with appropriate contrast

## Voice & Tone

### Voice Characteristics
- **Professional**: Expert, authoritative

## Naming Conventions

### Code
- **Functions**: `uvhttp_module_action` (snake_case)
- **Types**: `uvhttp_name_t` (snake_case with _t suffix)
- **Constants**: `UVHTTP_UPPER_CASE` (UPPER_CASE)
- **Macros**: `UVHTTP_UPPER_CASE` (UPPER_CASE)

### Documentation
- **Files**: `kebab-case.md`
- **Sections**: `Title Case`
- **Code blocks**: Use appropriate language tags
- **Links**: Descriptive anchor text

### Examples
- **Files**: `snake_case.c`
- **Classes**: `PascalCase`
- **Variables**: `snake_case`
- **Constants**: `UPPER_CASE`

### Code Snippets
```c
// Show complete example
#include <uvhttp.h>

int handler(uvhttp_request_t* req, uvhttp_response_t* res) {
    uvhttp_response_set_status(res, 200);
    uvhttp_response_set_header(res, "Content-Type", "application/json");
    return uvhttp_response_send(res);
}
```
